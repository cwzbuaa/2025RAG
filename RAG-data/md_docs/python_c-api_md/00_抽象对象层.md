或所有序列类型）。当使用对象类型并不适用时，他们会产生一个 Python 异常。
这些函数是不可能用于未正确初始化的对象的，如一个列表对象被 PyList_New() 创建，但其中的
项目没有被设置为一些非 NULL 的值。
